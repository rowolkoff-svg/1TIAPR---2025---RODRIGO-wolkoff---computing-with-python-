## Exercícios

- `ex01.py` – Número positivo ou negativo  
- `ex02.py` – Par ou ímpar  
- `ex03.py` – Maioridade  
- `ex04.py` – Nota de aprovação  
- `ex05.py` – Comparação de dois números

- `ex09.py` – Conversão de temperatura  
- `ex10.py` – Calculadora simples
